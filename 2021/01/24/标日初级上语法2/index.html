第二课

①これ／それ／あれ　は　N　です

意思: 这，这个/那，那个/那，那个 是N

例句

これ は 本 です。(这是书)
それ は 本 です。(那是书)
あれ は 本 です。(那是书)

※これ、それ、あれ区别

これ:离说话人近
それ:离听话人近
あれ:离两人都远

②誰(だれ)／何(なん)(特殊疑问词)

意思: 谁/什么

例句

それは 何 ですか？(那是什么？)(特殊疑问句)
あの人は 誰 ですか？(那个人是谁？)(特殊疑问句)
掌握：疑问句中有特殊疑问词就是特殊疑问句，回答特殊疑问句不能用はい、いいえ
了解：どなた 是 誰 的敬体

※回答(特殊疑问句)

回答それ用これ, 回答これ用それ, 回答あれ用あれ, 回答どれ根据情况使用こ/そ/あれ

例句

それ は何ですか？
これ は鉛筆です。

③この／その／あの + A(N) は　B(N)　です

意思: 这个/那个/那个 A 是B

例句

このカメラは森さんのです。(这个照相机是小森的)
そのカメラは森さんのです。(那个照相机是小森的)
あのカメラは森さんのです。(那个照相机是小森的)

小贴士

※ この + A(N) = これ(对于 A(N) 来说, 位置关系相同)
以上例句也可以变成这样：
このカメラは森さんのです。(这个照相机是小森的) = これは森さんのです。(这个照相机是小森的)
そのカメラは森さんのです。(那个照相机是小森的) = それは森さんのです。(那个照相机是小森的)
あのカメラは森さんのです。(那个照相机是小森的) = それは森さんのです。(那个照相机是小森的)

④どれ/(どの + N)

意思: 哪个

用途：必须是在有三个以上的事物中，不确定是哪个时才可以使用

例句

森さんの机はどれですか？(小森的桌子是哪个？)
森さんの机はどの机ですか？(小森的桌子是哪一张？)
这里也适用于知识点③的小贴士,上面两个句子也是等价的, 实际生活中使用那个都可以。

⑤の

意思: 的

例句

これは董さん の 本です。(这是小董的书)
